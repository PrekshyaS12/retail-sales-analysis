# Retail Sales Analysis — PostgreSQL

## Overview
Analyzed 9,994 retail transactions from the Superstore dataset 
to identify revenue trends, top customers, and regional performance.

## Tools
- PostgreSQL
- pgAdmin 4

## Queries
1. Total revenue by product category
2. Top 10 customers by total spend
3. Monthly sales trend
4. Running total of sales using window function
5. Profit by region
6. Month-over-month sales drop detection using LAG()

## Key Insights
- Technology is the highest revenue category
- West region generates the highest profit
- Sales peak in Q4 every year
